                 
Startup
--------
1. --> 1st terminal Start database  
[ContactDB]$ sudo mongod --dbpath /pathto/public_html/ContactDB/data  

2. --> 2nd terminal Start application server  
[ContactDB]$ node app.js  

3. --> Web browser  
http://localhost:3000/
