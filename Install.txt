
Express (MVC) / Mongo (DB) / Jade (in lieu of HTML) / Stylus (in lieu of css)

Environment Setup  for Project
------------------------------

npm install express

To automatically build out an application template for your application, navigate to the directory 
where you wish to create your application and run the following code:

mkdir ContactDB
cd ContactDB
express -c stylus
npm install -d

Add dependencies to package.json
--------------------------------

[ContactDB]$ vi package.json


Edit  package.json
add
, "mongodb": ">= 0.9.6-7"

Once you have saved the file run the following code to download and install the nodejs client for mongoDB:

npm install -d
